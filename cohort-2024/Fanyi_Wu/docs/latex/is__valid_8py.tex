\doxysubsection{Detailed Description}
Module containing a function for validating a number placed in a cell. 

The is\+\_\+valid() function determines the validity of placing a specific number \textquotesingle{}num\textquotesingle{} in the \textquotesingle{}grid\textquotesingle{} at position (\textquotesingle{}row\textquotesingle{}, \textquotesingle{}col\textquotesingle{}) The function executes a series of checks to assess the validity of placing a number in a Sudoku grid to ensure it follows the Sudoku rules\+:
\begin{DoxyItemize}
\item Number must be between 1 and 9
\item Number must not already exist in the given row
\item Number must not already exist in the given column
\item Number must not already exist in the 3x3 subgrid that includes the cell at (\textquotesingle{}row\textquotesingle{}, \textquotesingle{}col\textquotesingle{})
\end{DoxyItemize}
