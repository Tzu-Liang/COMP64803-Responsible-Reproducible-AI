\doxysubsection{Detailed Description}
Module containing a function for validating Sudoku grids. 

This module contains functions for validating the structure and correctness of Sudoku grids. It includes a function that checks whether a given grid is a valid 9x9 Sudoku grid, ensuring that each cell contains an integer between 0 and 9 and that there are no duplicates in rows, columns, or 3x3 subgrids, except for the number 0.
