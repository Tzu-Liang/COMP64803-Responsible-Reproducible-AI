grid_file.read_grid_file ( file_path)

Read a Sudoku puzzle from input file and convert it into a 2D list.

This function opens a file containing a Sudoku puzzle and parses each line into a list of integers, representing the Sudoku grid. This is achieved by calling parse_sudoku_line(line).

Parameters
file_pathThe path to the file containing the Sudoku puzzle.
Returns
A 2D list representing the Sudoku grid, or None if there is an error in file reading or if the puzzle format is incorrect.
Exceptions
FileNotFoundErrorIf the specified file cannot be found.
ValueErrorIf any row in the Sudoku puzzle is not of length 9.
ExceptionFor any other errors that occur during file reading.

Example usage: To read a Sudoku grid from 'input.txt':

grid = read_grid_file('input.txt')

The input.txt file should only include one Sudoku puzzle and in the format below:

$ cat input.txt
000|007|000
000|009|504
000|050|169
---+---+---
080|000|305
075|000|290
406|000|080
---+---+---
762|080|000
103|900|000
000|600|000

grid_file.save_grid_file ( grid,
file_path )

Save a Sudoku puzzle to a file from a 2D list representation.

This function writes a Sudoku grid to an output file, making the output Sudoku in the same format as the input file. It is achieved by calling format_sudoku_row(row)

  • converts each row into a string format with '|' separators for every three numbers.
  • After every third row (except the last one), a separator line '—+—+—' is added.
  • The function handles any exceptions that may occur during file writing.
Parameters
gridA 2D list representing the Sudoku grid.
file_pathThe path where the Sudoku puzzle will be saved.
Exceptions
ExceptionIf an error occurs while writing to the file.

Example usage: To save a Sudoku grid to 'output.txt':

save_grid_file(grid, 'output.txt')

The output.txt file will be in the same format as the input.txt file.
