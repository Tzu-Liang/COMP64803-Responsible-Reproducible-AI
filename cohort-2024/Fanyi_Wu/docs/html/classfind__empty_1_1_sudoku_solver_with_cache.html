Initializes the cache of empty cells in the grid.

The cache of empty cells is generated by calling the _initialize_empty_cells_cache() method. The cache of empty cells will be updated whenever a cell is updated in the grid by the update_cell() method.

Parameters
gridA Sudoku grid (2D list produced from read_grid_file()).

◆ find_empty()

find_empty.SudokuSolverWithCache.find_empty ( self)

Finds the next empty cell in the Sudoku grid using the cache.

Utilizing the aforementioned cache, this method quickly locates the next empty cell in the grid. It returns a tuple containing the row and column indices of the found empty cell, or None if the grid is fully occupied.

Returns
Tuple of (int, int): The row and column indices of the empty cell, or None if no empty cell is found.
